Geisel Library is a text adventure set in side of the UCSD campus, where you as a player is exploring the Geisel Library after hours. There has been a rumour that there is a secret chamber within the library and a deadly snake guarding it. Most people thinks this is a myth from referrencing the Harry Potter book series, but you are curious and decided to seek the truth. I chose this topic because Geisel does give off spooky and mustyvibe somethings, it'd be fun to make it even spookier.
